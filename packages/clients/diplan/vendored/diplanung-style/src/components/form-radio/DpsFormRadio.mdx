# DpsFormRadio

## Description

The `DpsFormRadio` component renders a basic radio input, with its label provided through the
default slot.

<Primary name="Default" />

## Stories

### Checked

To set the checkbox to **checked**, ensure that the `modelValue` matches the `value` property. By
default, the `value` property is `true`.

<Canvas of={DpsFormRadioStories.Checked} />

### Disabled

To disable the input element, set the `disabled` property to `true`.

<Canvas of={DpsFormRadioStories.Disabled} />

#### Checked

<Canvas of={DpsFormRadioStories.DisabledChecked} />

### Readonly

To disable the input element and display it as read-only, set the `readonly` property to `true`.

**Note:**

According to the HTML specification, there is no `readonly` attribute for input radios. Therefore,
when the `readonly` property is added, the `disabled` attribute is also applied to ensure the user
cannot alter the state of the radio. This approach prevents interaction while maintaining different
styling compared to a standard disabled state.

<Canvas of={DpsFormRadioStories.Readonly} />

To truncate the label, set the `truncate` property to `true`. When this property is set, ellipses
