# DpsFormDatepicker

## Description

The `DpsFormDatepicker` component is a customizable date picker designed for selecting single or
range dates. It supports various configurations, including optional checkboxes, blacklisted dates,
and clearable fields. The component is using the `vue-datepicker-next` npm package for its calendar
functionality.
