# DpsInputGroup

## Description

The `DpsInputGroup` component groups input elements together. It provides multiple slots, which are
displayed side by side.

The default example groups an icon, a `DpsFormInput` and a `DpsButton` together.

<Primary name="Default" />

## Stories

### Collapsed

To remove the gap between the slot content elements, set the `collapsed` property to `true`.

### Multiple Buttons
