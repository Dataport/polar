# DpsFormCheckboxGroup

## Description

The `DpsFormCheckboxGroup` component can be used to display a group of `DpsFormCheckbox` components.

To configure the options, add them to the `options` property:

```json
[
  { "label": "Option 1", "value": "opt-1" },
  { "label": "Option 2", "value": "opt-2" },
  { "label": "Option 3", "value": "opt-3" }
]
```

### Disabled Option

To disable individual checkboxes, add `disabled: true` to the objects within the array set in the
`options` property.

<Canvas of={DpsFormCheckboxGroupStories.DisabledOption} />

### Readonly Option

To render individual checkboxes as read-only, add `readonly: true` to the objects within the array
set in the `options` property.

<Canvas of={DpsFormCheckboxGroupStories.ReadonlyOption} />

### Error Option

To render individual checkboxes in their error state, add `error: true` to the objects within the
array set in the `options` property.

<Canvas of={DpsFormCheckboxGroupStories.ErrorOption} />

### Truncated Labels

To truncate the labels, set the `truncate` property to `true`.

<Canvas of={DpsFormCheckboxGroupStories.TruncatedLabels} />

### Object Values

This example demonstrates how to change the object keys for **value** and **label** using the
`optionLabel` and `optionValue` properties. In the provided options array, each object contains
multiple values, but only `value1` is used as the label and `value2` as the value.

<Canvas of={DpsFormCheckboxGroupStories.ObjectValues} />

### Inline
