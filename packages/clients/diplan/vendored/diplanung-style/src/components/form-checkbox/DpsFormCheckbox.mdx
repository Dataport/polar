# DpsFormCheckbox

## Description

The `DpsFormCheckbox` component renders a basic checkbox, with its label provided through the
default slot.

<Primary name="Default" />

## Stories

### Checked

To set the checkbox to **checked**, ensure that the `modelValue` matches the `value` property. By
default, the `value` property is `true`.

To disable the input element, set the `disabled` property to `true`.

### Readonly

To disable the checkbox and display it as read-only, set the `readonly` property to `true`.

**Note:**

According to the HTML specification, there is no `readonly` attribute for input checkboxes.
Therefore, when the `readonly` property is added, the `disabled` attribute is also applied to ensure
the user cannot alter the state of the checkbox. This approach prevents interaction while
maintaining different styling compared to a standard disabled state.
