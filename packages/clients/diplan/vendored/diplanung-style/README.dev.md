### Development

In case you want to modify the project files or run Storybook locally:

1. Clone the repository
2. Open a new terminal and navigate to the root of the cloned repository
3. Install all dependencies `npm install`
4. Start the dev environment `npm run storybook`
5. Open Storybook: [http://localhost:6006](http://localhost:6006/)

_**Requirements:**_

- `nodejs` >= 18 (LTS: 18.13.0)
- `npm` >= 8 (LTS: 8.19.3)
