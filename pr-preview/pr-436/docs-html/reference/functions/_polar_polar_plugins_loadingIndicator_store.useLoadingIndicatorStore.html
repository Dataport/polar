    addLoadingKey

    addLoadingKey: (key: string) => void

    Adds a loading indicator with the given key.

    The key is a unique identifier used to keep track of the added loader via a Set. It can't be added multiple times, and removing it once always removes it altogether.

    The LoadingIndicator will usually be used for asynchronous code.

    Remarks

    It is advised to use a key like {my-plugin-or-application-name}-{procedure-name} to avoid name conflicts.
